# InventoryTracker
Inventory Tracker made using Java 

A simple Java inventory management application using JOptionPane dialogs.

Keep track of your small business stock with an interactive and easy-to-use app!

** Features **
-----------------------
Add items with quantity

Update existing items

Remove items

View all items in the inventory

Low stock alerts (items with quantity < 5)

Search for specific items

Export inventory to a .txt file

** Built using **
-----------------
Java 
JDK 21 
Apache Netbeans IDE 

** Future Improvements **
--------------------------------

GUI with JFrame instead of dialogs

Save/load inventory automatically on app start/exit

Connect to a database for persistent storage

Support for CSV or Excel exports
